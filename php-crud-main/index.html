Validação por REGEX

Nome:

*Seu nome deverá iniciar com as letras E ou A seguidas de 5 números

Senha:

*Sua senha deve conter, uma letra maiúscula, uma letra minúscula e um número

Placa do carro:

*Sua placa deve estar no seguinte formato: xxx-yyyy

Telefone:

*Seu telefone deve estar no seguinte formato: (xx)9xxxx-xxxx ou 9xxxx-xxxx
